# Header Skeleton Reference

The canonical H1–H4 template for Get Drivers Ed state course pages.

## State Course Page (Product Page) — The 6-H2 Master Template

```
H1: [Primary Keyword — exact match]
    Example: "New York 5-Hour Online Pre-Licensing Course"

    (Above-fold UI strip — these are <p> NOT <h2>):
      <p> See What Our Students Are Saying!
      <p> Bundle & Save: Enhance Your Learning Experience!

    H3: Overview
        [Intro paragraph — primary KW in first sentence]
        [8–10 benefit bullets]

    H3: Outcomes
        H4: Complete the [Course Name] with Get Drivers Ed
            [2–3 sentence description]
        H4: Practice with Your Learner's Permit
            [2–3 sentence description]
        H4: Schedule and Pass the Road Test
            [2–3 sentence description]
        H4: Submit Required Documentation
            [2–3 sentence description]
        H4: Pay the License Fees
            [2–3 sentence description]
        H4: Receive Your License
            [2–3 sentence description]

    H3: Eligibility
        [Short intro to eligibility section]

H2: Who Can Take the [DMV/BMV + Duration + Course Name for State]?
    Example: "Who Can Take the DMV 5 Hour Class Online for New York?"
    [3-bullet eligibility list]

H2: Shedding More Light on This Course
    [3–4 paragraphs expanding with LSI terms, halo keywords]

H2: Recommended Courses to Enhance Your Learning
    [Internal link grid — 3–6 related state/course cards]

H2: Your Step-By-Step Guide to Getting Licensed
    H3: 1. Sign Up
        [2-sentence description]
    H3: 2. Get Your Learner's Permit
        [2-sentence description]
    H3: 3. Complete Practice Hours & Road Test Prep
        [2-sentence description]
    H3: 4. Earn Your [State] Driver's License
        [2-sentence description]
    [CTA link: "Review the [State] Course"]

H2: Why Should You Opt for This Course?
    [2-paragraph trust-build block]
    [Mention DMV/BMV approval]
    [Mention completion certificate mechanics]

H2: Frequently Asked Questions
    H3: [Q1 — uses primary KW, addresses eligibility]
        [A: 40–80 words, leads with direct answer]
    H3: [Q2 — uses secondary KW, addresses online/format]
        [A]
    H3: [Q3 — uses secondary KW, addresses duration]
        [A]
    H3: [Q4 — uses secondary KW, addresses coverage/topics]
        [A]
    H3: [Q5 — addresses legitimacy/approval]
        [A]
```

## Header Rules

1. **H1** — Exact primary KW, once per page, no brand suffix, not a question
2. **H2** — 6 of them, each with distinct search intent (Who / What / Links / How / Why / FAQ)
3. **H3** — Primary KW or secondary KW in FAQ H3s. Nested under H2s. Used for step lists.
4. **H4** — Only inside Outcomes/process lists. Never standalone.
5. **Never promote promo strips to H2** ("See What Our Students Are Saying!", "Bundle & Save") — these are `<p>` tags. NP|accel explicitly demoted these on the NY page.

## Intent Matrix (which H2 serves which search intent)

| H2 | Intent Served | Keyword Slot |
|---|---|---|
| Who Can Take | Eligibility check | Secondary KW |
| Shedding More Light | Topical expansion | 2–3 secondary + 3–4 LSI |
| Recommended Courses | Internal linking | None needed |
| Step-By-Step Guide | Process/how | State-specific LSI |
| Why Should You Opt | Trust/value prop | 1 primary + 1–2 authority LSI |
| FAQ | Objection handling | Primary + secondary spread across Qs |

    H2: [Subtopic 1 — halo KW]

## State-Agency Acronym Reference

Match the state's actual agency name in headers:

| State | Agency | Use In Headers |
|---|---|---|
| NY | DMV | "DMV" / "New York DMV" |
| OH | BMV | "BMV" / "Ohio BMV" |
| PA | PennDOT | "PennDOT" |
| FL | DHSMV | "FHSMV" (common) or "DMV" |
| CA | DMV | "DMV" / "California DMV" |
| TX | DPS | "Texas DPS" |
| WA | DOL | "DOL" / "Washington DOL" |
| OR | DMV (ODOT) | "DMV" |
| AZ | MVD | "MVD" / "Arizona MVD" |
| IL | SOS | "Secretary of State" |
| MA | RMV | "RMV" / "Massachusetts RMV" |

Getting this right is an E-E-A-T signal — using "DMV" on an Ohio page tells Google (and users) you don't know Ohio.
